# 시작하기

## 필요 조건
- `JDK 1.8` or higher
- `JUnit5`
- (선택적) 코틀린을 사용하는 경우 [autoparams-kotlin](./extensions/autoparams-kotlin.mdx)

## 설치하기

### Maven

Maven을 사용한다면, `pom.xml`에 다음 종속성을 추가하세요.

```xml
<dependency>
  <groupId>io.github.autoparams</groupId>
  <artifactId>autoparams</artifactId>
  <version>11.3.1</version>
</dependency>
```

### Gradle (Groovy)

Gradle Groovy DSL을 사용한다면, `build.gradle`에 다음 종속성을 추가하세요.

```gradle
testImplementation 'io.github.autoparams:autoparams:11.3.1'
```

### Gradle (Kotlin)

Gradle Kotlin DSL을 사용한다면, `build.gradle.kts`에 다음 종속성을 추가하세요.

```gradle
testImplementation("io.github.autoparams:autoparams:11.3.1")
```
