# 컬렉션과 배열

AutoParams는 컬렉션과 배열을 자동으로 생성할 수 있습니다. 기본적으로 생성되는 컬렉션과 배열의 크기는 3이지만, [`@Size`](https://javadoc.io/static/jakarta.validation/jakarta.validation-api/3.0.2/jakarta/validation/constraints/Size.html) 애너테이션을 사용하여 크기를 재정의할 수 있습니다.

예를 들어, 다음 테스트는 `@Size(min = 5)` 제약 조건이 적용될 때 AutoParams가 정확히 5개의 요소를 가진 `ArrayList<String>`와 `String[]`을 생성하는지 확인합니다.

<JavaKotlinCodeBlock>
```java
@Test
@AutoParams
void testMethod(@Size(min = 5) ArrayList<String> arrayList) {
    assertThat(arrayList).hasSize(5);
}
```

```kotlin
@Test
@AutoKotlinParams
fun testMethod(@Size(min = 5) arrayList: ArrayList<String>) {
    assertThat(arrayList).hasSize(5)
}
```
</JavaKotlinCodeBlock>

<JavaKotlinCodeBlock>
```java
@Test
@AutoParams
void testMethod(@Size(min = 5) String[] array) {
    assertThat(array).hasSize(5);
}
```

```kotlin
@Test
@AutoKotlinParams
fun testMethod(@Size(min = 5) array: Array<String>) {
    assertThat(array).hasSize(5)
}
```
</JavaKotlinCodeBlock>

이를 통해 테스트 코드를 깔끔하고 간결하게 유지하면서 실제 데이터 크기로 작업할 수 있습니다.
